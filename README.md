##提督很忙的拓展插件(KanColleViewer Plugins)
==========

KanColleViewer 在3.0版本之后增加了插件功能，可以通过添加插件完善kcv的功能。

本插件在只kcv3.4下测试，其他旧版本理论上可以，不过没有测试，使用时最好是用最新的kcv。


###主要功能

* 添加到系统托盘
* 最小化时隐藏任务栏上的kcv
* 退出kcv时提示
* 热键（老板键）
* 更换kcv窗体主题
* 设置kcv的工具面板位置
* 简单的舰队信息


### 使用方法

* 将解压出来的dll放到kcv的Plugins目录下然后重启kcv，如果失败的话右键dll属性看看有个没有个什么什么锁定的 把那个解锁掉再重启kcv
